راهنمای جامع اتصال هوشمند
چالش فاصله دارید؟ اگر فاصله سنسورها بیش از ۱۰۰ متر است، از ماژول‌های Lora (رنج ۱-۱۰ کیلومتر) استفاده کنید.

برای مانیتورینگ سالن‌ها، شما به یک برد ESP32 و سنسور DHT22 (دقیق‌تر از DHT11) نیاز دارید. این قطعات را می‌توانید از فروشگاه‌های الکترونیک تهیه کنید. هزینه راه‌اندازی هر سالن بسیار پایین است.

کامپیوتری که این برنامه روی آن نصب است و بردهای ESP32 باید به یک مودم وای‌فای مشترک متصل باشند. ابتدا آدرس IP کامپیوتر خود را پیدا کنید (در ویندوز دستور ipconfig). مثال: 192.168.1.15

سخت‌افزار باید داده‌های خود را به آدرس‌های زیر ارسال کند:

POST /finance/api/sensors/update
(برای دما و رطوبت)
POST /finance/api/iot/weight
(برای باسکول دیجیتال)
نمونه کد آماده (Arduino/C++):
// قطعه کد ارسال دیتا
HTTPClient http;
http.begin("http://[SERVER_IP]:5000/finance/api/sensors/update");
http.addHeader("Content-Type", "application/json");

// داده‌ها به صورت JSON
String json = "{\"pen_id\":1, \"temperature\":28.5, \"humidity\":55.2}";
int responseCode = http.POST(json);

if (responseCode > 0) {
    Serial.println("اتصال موفق! پاسخ سرور: " + String(responseCode));
}
http.end();
